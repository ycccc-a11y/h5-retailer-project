# 手机测试说明

## 前提条件
1. **手机和电脑连接在同一个WiFi网络**（重要！）
2. 确保电脑防火墙允许5001端口访问
3. 后端API服务已启动（运行 `start_api.bat`）

## 步骤

### 1. 启动后端API服务
双击运行 `start_api.bat`，确保看到：
```
许可证查询API服务器启动，监听端口 5001
```

### 2. 启动前端服务器
在项目目录下打开命令行，运行：
```bash
# Python 3
python -m http.server 8080

# 或者使用 Python 2
python -m SimpleHTTPServer 8080
```

### 3. 在手机上访问
打开手机浏览器（推荐Chrome），访问：
```
http://192.168.1.35:8080
```

### 4. 自动适配（无需手动配置）
✅ **代码已自动适配！** 无需手动修改任何配置
- 当你在电脑上访问 `localhost:8080` 时，自动使用 `localhost:5001`
- 当你在手机上访问 `192.168.1.35:8080` 时，自动使用 `192.168.1.35:5001`

## 注意事项
1. ✅ **IP地址自动适配**：如果IP地址变化，只需用新的IP地址访问即可，无需修改代码
2. 确保手机和电脑在同一WiFi网络
3. 如果无法访问，检查电脑防火墙设置（允许5001和8080端口）
4. 高德地图API Key需要在控制台配置允许的域名/IP（如果有限制的话）
5. 如果遇到CORS错误，确保后端API已经设置了正确的CORS头

## 获取电脑IP地址
如果IP地址不是192.168.1.35，可以在命令行运行：
```bash
ipconfig | findstr IPv4
```
